### 3.3.1 MVT 패턴

MVC(Model-View-Controller) 패턴
* Model: 데이터
* View: 사용자 인터페이스
* Controller: 데이터를 처리하는 로직
* MVC를 구분하여 한 요소가 다른 요소들에게 영향을 주지 않도록 설계하는 방식
* UI 디자이너 - 개발자의 업무 구분이 가능하다.
